HypeReca: Hybrid Partitioned Embeddings for Recommendor Model Training Acceleration
===

This repository includes the core code of ATC'25 paper
[SpaRes: Distributed Heterogeneous In-Memory Embedding Database for Training Recommender Models](https://www.usenix.org/conference/atc25/presentation/he-jiaao).

The project was originally called _DrESS: Distributed Embedding Serving System_.
We currently keep the name dress in our code.

### Usage

Use cmake to build this project into `libdress.so`.
A [variant of HugeCTR](https://github.com/laekov/hugectr-hypereca) is an example of using HypeReca.
